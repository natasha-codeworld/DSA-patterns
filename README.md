# DSA Patterns

A collection of Data Structures & Algorithms problems organized by patterns rather than platforms.

The goal of this repository is not just to solve problems, but to understand the underlying patterns that can be reused across multiple problems.

## Current Topics

### Trees

| Problem | Pattern |
|----------|----------|
| Lowest Common Ancestor | Recursive DFS |
| Maximum Width of Binary Tree | BFS Level Order Traversal |
| Nodes at Distance K | Tree to Graph Transformation |
| Burn Tree | Multi-Source BFS |

---

## Why this repository?

While solving DSA problems, I realized that many seemingly different questions are built upon a small set of reusable ideas.

This repository focuses on:

- Problem-solving patterns
- Time & Space complexity analysis
- Design thinking behind solutions
- Reusable approaches
